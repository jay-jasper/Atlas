# Atlas Logo Concepts

These are first-pass black-and-white SVG explorations for the approved Atlas logo direction.

Direction held constant across all three concepts:

- Offset Flow
- symbol-first, weak `A` reading
- medium central negative space
- blunt forward projection
- open silhouette
- line-and-plane hybrid

Files:

- `atlas-logo-concept-1-tight-core.svg`
  - tighter center, stronger structural lock
- `atlas-logo-concept-2-balanced-offset.svg`
  - recommended baseline, balanced negative space and forward motion
- `atlas-logo-concept-3-open-flow.svg`
  - more open center, lighter and more atmospheric

These files are monochrome on a transparent background so they can be inspected in light or dark contexts.
