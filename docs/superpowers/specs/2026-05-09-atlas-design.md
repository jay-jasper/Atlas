# Atlas: AI-Native Desktop Layer (2026 终极版)

## 1. 产品愿景 (Product Vision)
**Atlas** 是 AI 时代的“个人桌面底层系统”。它通过一个极简的菜单栏图标，集成了原本需要 10+ 个独立软件才能完成的功能。采用“本地模块化”设计，程序内置功能选配界面，用户可以根据需要自由开启或关闭特定的功能模块（如监控、截图、Agent 等），确保软件运行环境始终保持轻量化（10MB - 30MB 基础内存占用）。

---

## 2. 功能零件库 (PRD - Component Library)

### 🛠️ 第一类：系统性能与硬件感知 (Monitoring)
*   **实时监控**: CPU, 内存, 网速, 磁盘 I/O。
*   **硬件健康**: 电池健康、实时功耗 (W)。
*   **Port Master**: 一键查看并杀死占用特定端口（如 8080/3000）的进程。
*   **TokenBar**: 实时监控 OpenAI/Claude API 消耗金额。
*   **AI 负载监控**: 显示本地模型 (Ollama/LM Studio) 对 NPU/GPU 的实时占用。

### 📸 第二类：数据流转与 AI 交互 (Data Flow)
*   **截图/录屏**: 参考 Shottr 与微信截图工具，支持选区截图、尺寸显示、选区移动/微调、标注、马赛克、复制/保存、钉图、滚动截屏、GIF 录制、OCR 文字自动提取。
*   **万能翻译**: 划词、截图、AI 沉浸式翻译。
*   **智能剪切板**: 无限历史、图片预览、正则搜索。
*   **Scratchpad**: Markdown 临时便签，支持 AI 一键总结。

### 🪟 第三类：窗口管理与工作空间 (Layout)
*   **可视化分屏**: 菜单栏 3x3 网格滑动布局。
*   **场景恢复 (Workspaces)**: 一键还原“编程模式”、“会议模式”下的窗口布局。
*   **窗口置顶**: 强制悬浮特定窗口。
*   **多屏快传**: 窗口在不同显示器间瞬移。

### ⚙️ 第四类：系统快捷开关 (Utility)
*   **防休眠**: 类似 Amphetamine 的系统唤醒控制。
*   **演示模式**: 一键隐藏图标、壁纸纯色化、静音通知。
*   **DDC/CI 控制**: 菜单栏调节外接显示器亮度。
*   **Hand Mirror**: 开会前一键预览摄像头画面。

### 🤖 第五类：AI Agent 扩展 (Advanced Agent)
*   **AI Skill 接口**: 自定义技能（例：截图后自动总结并发送邮件）。
*   **Privacy Pulse**: 实时监控摄像头、麦克风及剪切板的调用详情。
*   **自动化触发**: Shell/Python 脚本一键运行。

### 🧩 第六类：功能选配中心 (Feature Manager)
*   **本地选配 UI**: 在软件设置中提供一个直观的“零件库”界面。
*   **实时开关**: 用户勾选功能后，对应模块实时加载/卸载，无需重启。
*   **依赖管理**: 自动处理模块间的依赖（例如：开启“截图翻译”会自动提示开启“截图”和“翻译”模块）。

## 3. 技术实施方案 (Technical Design)

### 3.1 动态模块加载架构 (Dynamic Module System)
*   **单二进制 + 动态服务控制**: 采用单个核心二进制文件，各功能模块在逻辑上完全解耦。
*   **Lazy Initialization**: 只有用户在选配界面开启的功能，其相关的后台服务（如剪贴板监听、系统指标采集、热键挂钩）才会被初始化。
*   **按需资源分配**: 禁用的模块不占用任何 CPU 轮询或内存资源，确保 Atlas 在极简模式下运行极其轻快。

### 3.2 性能保障 (Performance)
*   **Native UI**: 彻底弃用 Electron。macOS 使用 SwiftUI，Windows 使用 WinUI 3。
*   **Zero-Copy Memory**: Rust Core 与 Native UI 之间使用高效的内存共享协议，确保监控类数据更新时 CPU 占用近乎 0。
*   **Rust Runtime**: 使用多线程异步模型，确保耗时操作（如截图处理、OCR）不阻塞 UI 线程。

### 3.3 隐私与安全 (Security)
*   **Local-First**: Timeline 与历史数据默认存储于本地 SQLite。
*   **Privacy Pulse**: 核心层自动拦截系统级的隐私调用，并反馈给用户，确保 Atlas 自身与外部插件的透明度。

## 4. 商业模式与分发
*   **基础版 (Base Pack)**: 系统监控 + 截图翻译 + 基础窗口管理（免费）。
*   **专业版 (Pro Pack)**: AI Agent Skill + Workspace 场景恢复 + TokenBar（按月或永久订阅）。
*   **社区版**: 支持用户分享自定义插件与 Workflow。
